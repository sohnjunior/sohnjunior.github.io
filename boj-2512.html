BOJ 2512 - 예산

sohnjunior 17 May 2020 on Boj

문제

백준 온라인 저지 - 2512번

풀이 과정

예산의 집행 가능 여부에 따라 최적의 상한을 결정해야하는 문제이다.
다만 입력의 크기가 크기 때문에 상한을 1씩 증가하며 탐색하는 것은 비효율적이라 다른 방법이 필요하다.
이때 우리가 고려해볼 수 있는 것은 예산 상한의 최솟값과 최댓값을 알 수 있기 때문에 이를 활용해서 이분 탐색 을 수행하는 것이다.
각 도시마다 무조건 보장해줄 수 있는 예산 금액은 전체 예산 / 도시의 수 이므로 이를 최솟값으로 하고 가장 큰 예산 요청값을 최댓값으로 하여 이분 탐색 을 하면 된다,

코드


import sys

N = int(input())
requests = list(map(int, sys.stdin.readline().split()))
deposit = int(input())


def possible(limit):
    temp = 0
    for req in requests:
        temp += min(req, limit)
    if temp <= deposit:
        return True
    else:
        return False


def solution():
    max_request = max(requests)
    total = sum(requests)
    # 예산 내에서 가능할 경우
    if total <= deposit:
        return max_request

    # 예산 내에서 불가능할 경우
    lo = deposit // N
    hi = max_request

    answer = 0
    while lo < hi:
        mid = (lo + hi) // 2

        if possible(mid):
            answer = mid
            lo = mid + 1
        else:
            hi = mid

    return answer


print(solution())
