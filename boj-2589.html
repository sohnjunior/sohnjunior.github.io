BOJ 2589 - 보물섬

sohnjunior 04 Jun 2020 on Boj

문제

백준 온라인 저지 - 2589번

풀이 과정

지도의 각 칸마다 BFS 탐색을 통해 최단 거리중 가장 긴 경로를 찾아내야 합니다.
이때 최악의 경우 2500번의 BFS 탐색을 수행하고 각 BFS 탐색은 네 방향으로 진행되므로 O(4N) 의 시간 복잡도를 가지고 있습니다.
따라서 해당 방법을 사용해도 시간 제한내에 풀이가 가능하다는 것을 알 수 있으므로 지도에 존재하는 각 육지마다 최단 거리중 가장 긴 값을 반환하도록 해 답을 구하였습니다.

코드


import sys
from collections import deque

R, C = list(map(int, sys.stdin.readline().split()))
board = []
for _ in range(R):
    line = list(sys.stdin.readline().strip())
    board.append(line)

dx = [0, 0, 1, -1]
dy = [1, -1, 0, 0]


def bfs(r, c):
    q = deque()
    visit = [[0]*C for _ in range(R)]

    q.append((r, c, 0))
    visit[r][c] = 1

    longest = 0
    while q:
        x, y, cost = q.popleft()
        longest = max(longest, cost)

        for i in range(4):
            nx, ny = x + dx[i], y + dy[i]
            if 0 <= nx < R and 0 <= ny < C:
                if not visit[nx][ny] and board[nx][ny] != 'W':
                    visit[nx][ny] = 1
                    q.append((nx, ny, cost+1))

    return longest


def solution():
    ans = 0
    for r in range(R):
        for c in range(C):
            if board[r][c] == 'L':
                ans = max(ans, bfs(r, c))
    return ans


print(solution())
